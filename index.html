Learning GraphQL? Practice with local GraphQL server in 5 simple steps:

  1. From the command line (in a folder of your choice), run git clone https://github.com/asa55/graphql-practice.git
  2. From VSCode (using the "Remote - Containers" extension), "Open Folder In Container". (This step requires Docker)
  3. From inside the container (VSCode's terminal works fine), run node index.js
  4. Using your favorite browser, visit localhost:4000
  5. Play with GraphQL! Looking for a place to start? Check out my GraphQL notes on GitHub here
